## 新型冠状病毒疫情地图

- 前端框架：[Create React App](https://github.com/facebook/create-react-app)
- 数据说明： 2019年12月1日——2020年4月18日 世界新冠肺炎累计确诊、治愈、死亡人数，中国精确到市区级，国外数据基本以国家为单位，个别国家精确到州。

`npm start` 以开发模式运行  
`npm test` 测试  
`npm run build` 构建
